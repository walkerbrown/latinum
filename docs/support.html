Getting Started

  1. Install Latinum from the App Store
  2. Open Settings → General → Keyboard → Keyboards
  3. Tap Add New Keyboard and select Latinum
  4. Switch to the Latinum keyboard in any app using the globe key 🌐

Tips

  • Long-press vowel keys for macrons (ā, ē, ī, ō, ū) and ligatures (æ, œ)
  • Tap a suggestion to complete or predict the next word
  • Select a word to see alternative completions
